# Currency Converter

A simple web-based currency converter built with HTML, CSS, and JavaScript.

## Features

- Convert between multiple currencies (USD, EUR, GBP, JPY, CAD, AUD, CHF, CNY, INR)
- Real-time exchange rates fetched from exchangerate-api.com
- Clean and responsive design
- Country flags displayed next to currency names for easy identification

## Usage

1. Open `index.html` in your web browser
2. Enter the amount you want to convert
3. Select the source currency from the "From" dropdown
4. Select the target currency from the "To" dropdown
5. Click the "Convert" button or press Enter
6. View the converted amount in the result area

## Files

- `index.html`: Main HTML structure
- `styles.css`: CSS styling
- `script.js`: JavaScript logic for conversion

## API

This application uses the free ExchangeRate-API (https://exchangerate-api.com/) to fetch current exchange rates.

## Troubleshooting

- If exchange rates don't load, check your internet connection
- Ensure JavaScript is enabled in your browser
- For local development, you may need to disable CORS restrictions or use a local server
