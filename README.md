# Calculator App

A simple and responsive Calculator application built using HTML, CSS, and JavaScript.  
This project performs basic arithmetic operations like addition, subtraction, multiplication, and division.

---

## 🚀 Features

- Basic arithmetic operations
  - Addition (+)
  - Subtraction (-)
  - Multiplication (×)
  - Division (÷)
- Clear button (C)
- Delete last character (DEL)
- Responsive design
- User-friendly interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
